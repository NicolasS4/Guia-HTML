Textos e Semântica

Elementos HTML para formatação e significado semântico de conteúdo textual.

30 min de leitura Atualizado em: 15/08/2025

Introdução à Semântica Textual

HTML5 oferece elementos específicos para adicionar significado semântico ao conteúdo textual, melhorando a acessibilidade, SEO e manutenibilidade.

Elementos semânticos descrevem o significado do conteúdo, não apenas sua aparência visual.

Elementos Semânticos de Texto

<a> - Link

Cria um hyperlink para outras páginas, arquivos, endereços de email ou qualquer URL.

<a href="https://exemplo.com" target="_blank" rel="noopener">Visite nosso site</a>

Atributos Importantes:

  • href - URL de destino
  • target - Onde abrir o link (_blank para nova aba)
  • rel - Relacionamento com o destino (noopener para segurança)
  • download - Força o download do recurso

<strong> - Importância Forte

Indica que o conteúdo tem forte importância, seriedade ou urgência. Normalmente renderizado em negrito.

<p><strong>Atenção:</strong> Esta operação não pode ser desfeita.</p>

Diferente de <b>, que apenas aplica negrito, <strong> tem significado semântico.

<em> - Ênfase

Indica ênfase em um texto. Normalmente renderizado em itálico.

<p>Eu <em>realmente</em> preciso terminar este projeto.</p>

Diferente de <i>, que apenas aplica itálico, <em> altera o significado da frase.

<mark> - Texto Destacado

Representa texto marcado ou destacado para fins de referência.

<p>Procure por <mark>palavras-chave</mark> no texto.</p>

<small> - Texto Secundário

Representa comentários laterais e texto pequeno, como informações de direitos autorais.

<p>Texto normal <small>com um comentário pequeno</small>.</p>

<del> e <ins> - Texto Removido/Inserido

Representam texto que foi removido (<del>) ou inserido (<ins>) no documento.

<p>O custo é de <del>R$ 100</del> <ins>R$ 80</ins>.</p>

<sub> e <sup> - Subscrito e Sobrescrito

Representam texto subscrito (<sub>) e sobrescrito (<sup>).

<p>H<sub>2</sub>O é a fórmula da água.</p>
<p>O teorema de Pitágoras: a<sup>2</sup> + b<sup>2</sup> = c<sup>2</sup>.</p>

Elementos para Código e Dados

<code> - Código Fonte

Exibe um fragmento de código de computador. Normalmente renderizado em fonte monoespaçada. 

<p>Use <code>console.log()</code> para imprimir no console.</p>

<kbd> - Entrada do Teclado

Representa entrada do usuário, como teclas do teclado ou comandos.

<p>Pressione <kbd>Ctrl</kbd> + <kbd>C</kbd> para copiar.</p>

<samp> - Saída de Amostra

Representa a saída de um programa ou sistema computacional.

<p>O programa exibiu: <samp>Arquivo não encontrado</samp></p>

<var> - Variável

Representa o nome de uma variável em expressões matemáticas ou contextos de programação.

<p>A equação da reta é <var>y</var> = <var>m</var><var>x</var> + <var>b</var>.</p>

Elementos para Datas e Abreviações

<time> - Data/Hora

Representa um período específico no tempo, podendo incluir data e hora.

<p>O evento ocorrerá em <time datetime="2025-12-31">31 de dezembro</time>.</p>
<p>Horário de abertura: <time datetime="09:00">9h</time> às <time datetime="18:00">18h</time>.</p>

Atributo:

  • datetime - Formato machine-readable da data/hora

<abbr> - Abreviação

Representa uma abreviação ou acrônimo, com explicação opcional.

<p>A <abbr title="World Wide Web">WWW</abbr> foi criada em 1989.</p>

Atributo:

  • title - Explicação da abreviação (exibida como tooltip)

Elementos de Formatação sem Semântica

<b> - Negrito

Texto em negrito sem importância semântica adicional.

<p>Este é um texto em <b>negrito</b>.</p>

Prefira <strong> quando o texto tiver importância semântica.

<i> - Itálico

Texto em itálico sem ênfase semântica adicional.

<p>O termo <i>Homo sapiens</i> é em latim.</p>

Prefira <em> quando o texto precisar de ênfase semântica.

<u> - Sublinhado

Texto sublinhado. Evite usar para links para não confundir usuários.

<p>Este texto contém um <u>erro ortográfico</u>.</p>

<br> - Quebra de Linha

Insere uma quebra de linha em um local específico.

<p>Primeira linha<br>Segunda linha</p>

Evite usar múltiplas <br> para criar espaçamento. Use CSS margin/padding.

<wbr> - Oportunidade de Quebra

Indica um ponto onde o navegador pode quebrar a linha se necessário.

<p>Esteéumpalavramuitogrande<wbr>quepodeprecisardequebra.</p>

Exemplo Prático

Veja como esses elementos podem ser usados juntos:

<article>
    <h1>Novidades no <abbr title="HyperText Markup Language 5">HTML5</abbr></h1>
    
    <p>
        O <strong>HTML5</strong> introduziu vários novos elementos semânticos 
        como <code>&lt;header&gt;</code>, <code>&lt;footer&gt;</code> e 
        <code>&lt;article&gt;</code> que <em>melhoram significativamente</em> 
        a estruturação de documentos web.
    </p>
    
    <p>
        Publicado em: <time datetime="2025-08-15">15 de agosto de 2025</time><br>
        Última atualização: <time datetime="2025-08-15T14:30">14:30</time>
    </p>
    
    <p>
        Para formatar código, use <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>F</kbd>. 
        O resultado será algo como:
    </p>
    
    <pre><code>
    function exemplo() {
        console.log("Olá, mundo!");
    }
    </code></pre>
    
    <p>
        <small>Este artigo foi escrito por John Doe e está licenciado sob CC BY-SA.</small>
    </p>
</article>
