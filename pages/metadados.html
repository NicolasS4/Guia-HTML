Metadados HTML

Elementos que fornecem informações sobre o documento HTML, não exibidos diretamente na página.

18 min de leitura Atualizado em: 15/08/2025

O que são Metadados?

Metadados são informações sobre o documento HTML que não são exibidas diretamente na página, mas são essenciais para navegadores, mecanismos de busca e outras aplicações que processam o documento.

Os metadados são colocados dentro do elemento <head> do documento HTML e incluem título, descrição, autor, palavras-chave e outras informações úteis.

Elementos de Metadados

<meta> - Metainformações

O elemento <meta> fornece metadados sobre o documento HTML, como charset, viewport, descrição e palavras-chave.

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Um guia completo sobre HTML5">
<meta name="keywords" content="HTML, HTML5, guia, tutorial">
<meta name="author" content="Seu Nome">

Atributos Comuns:

  • charset - Define a codificação de caracteres (UTF-8 é recomendado)
  • name - Especifica o tipo de metadado
  • content - Fornece o valor associado ao metadado
  • http-equiv - Usado para cabeçalhos HTTP simulados

<title> - Título do Documento

Define o título do documento, exibido na aba do navegador e nos resultados de busca.

<title>Meu Site Incrível</title>

Boas Práticas:

  • Mantenha títulos curtos e descritivos (50-60 caracteres)
  • Inclua palavras-chave relevantes no início
  • Evite títulos genéricos como "Home" ou "Página 1"

<link> - Ligações a Recursos Externos

Usado para vincular folhas de estilo, ícones e outros recursos externos.

<link rel="stylesheet" href="estilos.css">
<link rel="icon" href="favicon.ico" type="image/x-icon">
<link rel="preconnect" href="https://fonts.googleapis.com">

Atributos Importantes:

  • rel - Define o relacionamento com o documento
  • href - Especifica a URL do recurso
  • type - Indica o tipo MIME do recurso

<style> - CSS Interno

Permite incluir folhas de estilo CSS diretamente no documento HTML.

<style>
    body {
        font-family: Arial, sans-serif;
        margin: 0;
        padding: 20px;
    }
</style>

Para melhor desempenho e manutenibilidade, prefira usar arquivos CSS externos com <link> para estilos mais complexos.

<script> - Código JavaScript

Usado para incluir código JavaScript, seja interno ou externo.

<script src="script.js"></script>
<script>
    console.log('Olá, mundo!');
</script>

Recomendações:

  • Use o atributo defer ou async para scripts externos quando apropriado
  • Coloque scripts no final do <body> para melhor desempenho
  • Para módulos JavaScript, use type="module"

<noscript> - Conteúdo Alternativo

Define conteúdo a ser exibido quando scripts estão desativados.

<noscript>
    <p>Seu navegador não suporta JavaScript ou está desativado.</p>
</noscript>

<base> - URL Base

Especifica a URL base para todos os links relativos no documento.

<base href="https://www.exemplo.com/" target="_blank">

Use com cautela, pois afeta todos os links relativos na página. Pode causar problemas se não configurado corretamente.

Metadados para SEO

Meta Tags Essenciais para Otimização de Busca

<meta name="description" content="Descrição clara e concisa do conteúdo da página">
<meta name="keywords" content="palavra-chave1, palavra-chave2, palavra-chave3">
<meta name="author" content="Nome do Autor">
<meta name="robots" content="index, follow">

<!-- Open Graph / Facebook -->
<meta property="og:title" content="Título da Página">
<meta property="og:description" content="Descrição para compartilhamento">
<meta property="og:image" content="URL da imagem para compartilhamento">
<meta property="og:url" content="URL da página">
<meta property="og:type" content="website">

<!-- Twitter -->
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Título para Twitter">
<meta name="twitter:description" content="Descrição para Twitter">
<meta name="twitter:image" content="URL da imagem para Twitter">
