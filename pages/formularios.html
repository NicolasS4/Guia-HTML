Formulários HTML

Elementos para criação de formulários interativos e acessíveis.

32 min de leitura Atualizado em: 15/08/2025

Elementos Básicos de Formulários

<form> - Formulário

Container para elementos de entrada que serão enviados para um servidor.

<form action="/processar" method="post" novalidate>
    <!-- Elementos do formulário -->
</form>

Atributos Importantes:

  • action - URL para enviar os dados
  • method - HTTP method (GET ou POST)
  • novalidate - Desativa validação nativa
  • autocomplete - Controla preenchimento automático

<input> - Campo de Entrada

Elemento versátil para vários tipos de entrada, dependendo do atributo type.

<input type="text" id="nome" name="nome" required>

Atributos Comuns:

  • type - Define o tipo de entrada (text, email, password, etc.)
  • name - Nome do campo para envio
  • id - Identificador único
  • required - Campo obrigatório
  • placeholder - Texto de exemplo
  • pattern - Expressão regular para validação

Tipos de <input>

email

<input type="email">

password

<input type="password">

<input type="checkbox">

<input type="radio">

<input type="file">

<textarea> - Área de Texto

Campo para entrada de texto multilinha.

<textarea id="mensagem" name="mensagem" rows="4" cols="50">
Texto padrão aqui
</textarea>

<select>, <option> e <optgroup>

Criam listas suspensas de opções.

<select id="cidade" name="cidade">
    <optgroup label="Sudeste">
        <option value="sp">São Paulo</option>
        <option value="rj">Rio de Janeiro</option>
    </optgroup>
    <option value="outra">Outra</option>
</select>

Atributos Úteis:

  • multiple - Permite seleção múltipla
  • size - Número de opções visíveis
  • selected - Opção pré-selecionada

<button> - Botão

Elemento clicável que pode enviar formulários ou acionar ações.

<button type="submit">Enviar</button>
<button type="reset">Limpar</button>
<button type="button">Ação</button>

Elementos de Apoio a Formulários

<label> - Rótulo

Associa um texto descritivo a um elemento de formulário, melhorando a acessibilidade.

<label for="nome">Nome Completo:</label>
<input type="text" id="nome" name="nome">

<!-- Ou -->
<label>
    <input type="checkbox" name="termos">
    Aceito os termos
</label>

<fieldset> e <legend>

Agrupam elementos relacionados e fornecem um título para o grupo.

<fieldset>
    <legend>Informações Pessoais</legend>
    <!-- Campos do formulário -->
</fieldset>

<datalist> - Lista de Sugestões

Fornece uma lista de opções pré-definidas para um campo <input>.

<input list="browsers" name="browser">
<datalist id="browsers">
    <option value="Chrome">
    <option value="Firefox">
    <option value="Edge">
</datalist>

<output> - Resultado de Cálculo

Exibe o resultado de um cálculo ou ação do usuário.

<form oninput="result.value = parseInt(a.value) + parseInt(b.value)">
    <input type="number" id="a" value="10"> +
    <input type="number" id="b" value="20"> =
    <output name="result" for="a b">30</output>
</form>

<progress> e <meter>

Exibem progresso e medições escalares.

<!-- Barra de progresso -->
<progress value="70" max="100">70%</progress>

<!-- Medidor -->
<meter value="6" min="0" max="10">6 de 10</meter>

Validação de Formulários

HTML5 introduziu validação nativa sem JavaScript:

Atributos de Validação

<!-- Campo obrigatório -->
<input type="text" required>

<!-- Comprimento mínimo e máximo -->
<input type="text" minlength="3" maxlength="20">

<!-- Valor mínimo e máximo -->
<input type="number" min="1" max="100">

<!-- Padrão regex -->
<input type="text" pattern="[A-Za-z]{3}">

<!-- Validação personalizada -->
<input type="text" oninvalid="this.setCustomValidity('Por favor, preencha corretamente')">

Use validação do lado do cliente

Para feedback imediato, mas sempre valide também no servidor.

Mensagens claras

Personalize mensagens de erro com setCustomValidity().

Desative autocomplete

Para campos sensíveis: autocomplete="off".

Exemplo Completo de Formulário

<form id="cadastro" action="/cadastro" method="post" novalidate>
    <fieldset>
        <legend>Informações Pessoais</legend>
        
        <div class="form-group">
            <label for="nome">Nome Completo:</label>
            <input type="text" id="nome" name="nome" required minlength="3">
        </div>
        
        <div class="form-group">
            <label for="email">E-mail:</label>
            <input type="email" id="email" name="email" required>
        </div>
        
        <div class="form-group">
            <label for="nascimento">Data de Nascimento:</label>
            <input type="date" id="nascimento" name="nascimento" required>
        </div>
    </fieldset>
    
    <fieldset>
        <legend>Preferências</legend>
        
        <div class="form-group">
            <label for="cor">Cor Favorita:</label>
            <input type="color" id="cor" name="cor" value="#ff0000">
        </div>
        
        <div class="form-group">
            <label>Interesses:</label>
            <label><input type="checkbox" name="interesses" value="tech"> Tecnologia</label>
            <label><input type="checkbox" name="interesses" value="esportes"> Esportes</label>
        </div>
        
        <div class="form-group">
            <label for="nota">Nota para nosso site:</label>
            <input type="range" id="nota" name="nota" min="1" max="10">
            <output for="nota" id="nota-valor">5</output>
        </div>
    </fieldset>
    
    <div class="form-actions">
        <button type="submit">Enviar</button>
        <button type="reset">Limpar</button>
    </div>
</form>

<script>
    document.getElementById('nota').addEventListener('input', function() {
        document.getElementById('nota-valor').value = this.value;
    });
</script>
