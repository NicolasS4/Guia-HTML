Imagens e Multimídia

Elementos HTML para incorporação de imagens, áudio, vídeo e outros conteúdos multimídia.

15 min de leitura Atualizado em: 15/08/2025

Elementos de Imagem

<img> - Imagem

Incorporar uma imagem no documento.

<img src="imagem.jpg" alt="Descrição da imagem" width="800" height="600" loading="lazy">

Atributos Essenciais:

  • src - Caminho para a imagem
  • alt - Texto alternativo (obrigatório para acessibilidade)
  • width/height - Dimensões (ajuda no layout)
  • loading - "lazy" para carregamento tardio

<picture> - Imagem Responsiva

Fornece versões alternativas de uma imagem para diferentes condições de exibição.

<picture>
    <source media="(min-width: 1200px)" srcset="imagem-grande.jpg">
    <source media="(min-width: 800px)" srcset="imagem-media.jpg">
    <img src="imagem-pequena.jpg" alt="Descrição">
</picture>

Use para art direction (imagens diferentes em diferentes tamanhos) ou formatos alternativos (WebP, AVIF).

Elementos de Áudio e Vídeo

<audio> - Áudio

Incorporar conteúdo de áudio no documento.

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Seu navegador não suporta o elemento de áudio.
</audio>

Atributos:

  • controls - Mostra controles de reprodução
  • autoplay - Inicia automaticamente (evitar)
  • loop - Reproduz em loop
  • muted - Inicia mudo

<video> - Vídeo

Incorporar conteúdo de vídeo no documento.

<video width="640" height="360" controls poster="thumbnail.jpg">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    Seu navegador não suporta o elemento de vídeo.
</video>

Atributos Adicionais:

  • poster - Imagem de pré-visualização
  • preload - Como pré-carregar o vídeo

Elementos de Suporte

<source> - Fonte Alternativa

Especifica múltiplos recursos de mídia para elementos <picture>, <audio> e <video>. 

<!-- Para picture -->
<source srcset="imagem.webp" type="image/webp">

<!-- Para audio/video -->
<source src="midia.ogg" type="audio/ogg; codecs=opus">

<track> - Legenda

Adiciona trilhas de texto (legendas, descrições) para elementos <audio> e <video>. 

<track 
    src="legendas.vtt" 
    kind="subtitles" 
    srclang="pt" 
    label="Português" 
    default
>

Atributos:

  • kind - Tipo de trilha (subtitles, captions, descriptions, etc.)
  • srclang - Idioma da trilha
  • label - Título para o usuário
  • default - Usar esta trilha por padrão

Outros Elementos Multimídia

<embed> - Conteúdo Externo

Incorporar conteúdo externo ou plug-in (como Flash, PDF viewers).

<embed 
    type="video/webm" 
    src="video.webm" 
    width="400" 
    height="300"
>

Evite usar quando possível. Prefira elementos nativos como <video> ou <iframe>.

<object> e <param>

Incorporar objetos externos com parâmetros configuráveis.

<object data="flash.swf" type="application/x-shockwave-flash">
    <param name="quality" value="high">
    Conteúdo alternativo para navegadores sem suporte.
</object>

<svg> - Gráficos Vetoriais

Container para gráficos vetoriais escaláveis (SVG).

<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" fill="red" />
</svg>

<canvas> - Área de Desenho

Área para renderização de gráficos via JavaScript.

<canvas id="meuCanvas" width="200" height="100">
    Seu navegador não suporta canvas.
</canvas>
<script>
    const canvas = document.getElementById('meuCanvas');
    const ctx = canvas.getContext('2d');
    ctx.fillStyle = 'red';
    ctx.fillRect(10, 10, 50, 50);
</script>

Boas Práticas para Multimídia

Textos Alternativos

Sempre forneça alt descritivo para imagens e transcrições para áudio/vídeo.

Formatos Modernos

Prefira formatos como WebP para imagens e WebM para vídeos quando possível.

Responsividade

Use <picture> e srcset para imagens responsivas.

Performance

Use loading="lazy" para imagens abaixo do dobra e compressão adequada.
