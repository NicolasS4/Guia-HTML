Agrupamento de Conteúdo

Elementos HTML para organizar e agrupar conteúdo em blocos lógicos.

15 min de leitura Atualizado em: 15/08/2025

Elementos de Agrupamento

HTML oferece vários elementos para agrupar conteúdo relacionado, criando estrutura e organização na página.

Elementos Básicos de Agrupamento

<div> - Divisão Genérica

Container genérico sem significado semântico, usado para agrupar conteúdo para fins de estilo ou manipulação via JavaScript.

<div class="container">
    <h2>Título</h2>
    <p>Conteúdo agrupado dentro de uma div.</p>
</div>

Prefira elementos semânticos quando possível. Use <div> apenas quando nenhum outro elemento for apropriado.

<span> - Container Inline

Container genérico inline sem significado semântico, usado para agrupar texto ou elementos inline para estilização.

<p>Este é um <span class="destaque">texto destacado</span> no parágrafo.</p>

<p> - Parágrafo

Representa um parágrafo de texto. Navegadores adicionam automaticamente margens antes e depois.

<p>Este é um parágrafo de exemplo.</p>
<p>Este é outro parágrafo separado do anterior.</p>

<hr> - Quebra Temática

Representa uma quebra temática entre elementos (como uma mudança de cena ou tópico). Exibido como uma linha horizontal.

<p>Conteúdo da primeira seção.</p>
<hr>
<p>Conteúdo da segunda seção, com tema diferente.</p>

<pre> - Texto Pré-formatado

Exibe texto exatamente como escrito no código HTML, preservando espaços em branco e quebras de linha.

<pre>
    function hello() {
        console.log("Hello, world!");
    }
</pre>

<blockquote> - Citação Longa

Representa uma citação extensa de outra fonte. Normalmente renderizado com indentação.

<blockquote cite="https://exemplo.com/fonte">
    <p>Esta é uma longa citação de outra fonte que deve ser destacada.</p>
</blockquote>

Atributo:

  • cite - URL da fonte da citação (opcional)

Listas

<ol> - Lista Ordenada

Lista numerada onde a ordem dos itens é importante.

<ol>
    <li>Primeiro item</li>
    <li>Segundo item</li>
    <li>Terceiro item</li>
</ol>

Atributos:

  • reversed - Inverte a numeração
  • start - Número inicial
  • type - Tipo de marcador (1, A, a, I, i)

<ul> - Lista Não Ordenada

Lista com marcadores onde a ordem dos itens não é importante.

<ul>
    <li>Item com marcador</li>
    <li>Outro item</li>
</ul>

<li> - Item de Lista

Representa um item em uma lista (<ol> ou <ul>).

<li>Item simples</li>
<li value="5">Item com valor específico (para ol)</li>

<dl> - Lista de Definições

Lista de termos e suas definições (como em um glossário).

<dl>
    <dt>HTML</dt>
    <dd>Linguagem de marcação para criar páginas web.</dd>
    
    <dt>CSS</dt>
    <dd>Linguagem para estilizar páginas web.</dd>
</dl>

Elementos Associados:

  • <dt> - Termo a ser definido
  • <dd> - Definição do termo

Elementos para Conteúdo Incorporado

<figure> e <figcaption>

Representam conteúdo autônomo (como imagens, diagramas, códigos) com uma legenda opcional. 

<figure>
    <img src="imagem.jpg" alt="Descrição">
    <figcaption>Legenda para a imagem</figcaption>
</figure>

Use para agrupar imagens com suas legendas ou blocos de código com descrições.

Exemplo Prático

Veja como esses elementos podem ser usados juntos:

<article>
    <h2>Receita de Bolo</h2>
    
    <figure>
        <img src="bolo.jpg" alt="Bolo de chocolate">
        <figcaption>Bolo de chocolate caseiro</figcaption>
    </figure>
    
    <div class="ingredientes">
        <h3>Ingredientes</h3>
        <ul>
            <li>2 xícaras de farinha</li>
            <li>1 xícara de açúcar</li>
            <li>3 ovos</li>
        </ul>
    </div>
    
    <div class="preparo">
        <h3>Modo de Preparo</h3>
        <ol>
            <li>Misture os ingredientes secos</li>
            <li>Adicione os ovos e misture bem</li>
            <li>Asse por 40 minutos</li>
        </ol>
    </div>
    
    <blockquote>
        <p>Esta receita foi passada de geração em geração em minha família.</p>
    </blockquote>
</article>
