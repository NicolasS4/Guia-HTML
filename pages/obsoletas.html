Tags Obsoletas no HTML5

Elementos HTML descontinuados e suas alternativas recomendadas.

20 min de leitura Atualizado em: 15/08/2025

Por que tags se tornam obsoletas?

O HTML5 removeu várias tags que eram usadas em versões anteriores do HTML, principalmente por:

  • Presentacionalismo: Tags que apenas definiam aparência, não significado
  • Má prática: Tags que incentivavam estruturas não semânticas
  • Tecnologias melhores: Tags substituídas por CSS ou novas alternativas semânticas
  • Problemas de acessibilidade: Tags que criavam barreiras para usuários com deficiência

Lista de Tags Obsoletas

Tag Obsoleta Descrição Original Alternativa Recomendada
<acronym> Definia um acrônimo <abbr> com atributo title
<applet> Incorporava applets Java <object> ou tecnologias modernas como JavaScript
<basefont> Definia estilos de fonte base CSS (font-family, font-size)
<big> Aumentava o tamanho do texto CSS (font-size)
<center> Centralizava conteúdo CSS (text-align: center ou margin: auto)
<dir> Lista de diretórios <ul>
<font> Definia fonte, cor e tamanho CSS (font-family, color, font-size)
<frame>, <frameset>, <noframes> Criava layouts com frames CSS Layout (Flexbox, Grid) ou <iframe> para conteúdo externo
<strike> Texto tachado <del> ou CSS (text-decoration: line-through)
<tt> Texto em fonte teletype <code> ou CSS (font-family: monospace)

Alternativas Detalhadas

<font> → CSS

O elemento <font> era usado para controlar aparência do texto diretamente no HTML.

Obsoleto:

<font face="Arial" size="4" color="red">
    Texto estilizado
</font>

Recomendado:

<span class="texto-destaque">
    Texto estilizado
</span>

<style>
    .texto-destaque {
        font-family: Arial;
        font-size: 1rem;
        color: red;
    }
</style>

<center> → CSS

O elemento <center> centralizava conteúdo horizontalmente.

Obsoleto:

<center>
    Conteúdo centralizado
</center>

Recomendado:

<div class="centro">
    Conteúdo centralizado
</div>

<style>
    .centro {
        text-align: center; /* Para texto */
        
        /* Para blocos: */
        margin-left: auto;
        margin-right: auto;
    }
</style>

<frameset> → Layouts Modernos

Framesets eram usados para dividir a janela em várias seções.

Obsoleto:

<frameset cols="25%,75%">
    <frame src="menu.html">
    <frame src="conteudo.html">
</frameset>

Recomendado:

<div class="layout">
    <nav>
        <!-- Conteúdo do menu -->
    </nav>
    <main>
        <!-- Conteúdo principal -->
    </main>
</div>

<style>
    .layout {
        display: grid;
        grid-template-columns: 25% 75%;
    }
</style>

Migrando para HTML5

Separe estrutura e apresentação

Use HTML para estrutura e significado, CSS para aparência.

Prefira elementos semânticos

Use <header>, <nav>, <article> em vez de <div> genéricos.

Valide seu código

Use o Validador W3C para identificar tags obsoletas.
