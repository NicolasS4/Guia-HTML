Tabelas HTML

Estruturas para exibição de dados tabulares de forma acessível e semântica.

20 min de leitura Atualizado em: 15/08/2025

Elementos Básicos de Tabelas

<table> - Tabela

Elemento raiz que define uma tabela.

<table>
    <!-- Conteúdo da tabela -->
</table>

Use tabelas apenas para dados tabulares, não para layout de página.

<tr> - Linha da Tabela

Define uma linha na tabela.

<tr>
    <!-- Células da linha -->
</tr>

<td> - Célula de Dados

Define uma célula padrão que contém dados.

<td>Conteúdo da célula</td>

Atributos:

  • colspan - Número de colunas que a célula ocupa
  • rowspan - Número de linhas que a célula ocupa
  • headers - Relaciona a célula com cabeçalhos

<th> - Célula de Cabeçalho

Define uma célula de cabeçalho que descreve uma coluna ou linha.

<th scope="col">Nome da Coluna</th>

Atributos Importantes:

  • scope - Define o escopo do cabeçalho (col, row, colgroup, rowgroup)
  • abbr - Versão abreviada do conteúdo

Elementos de Estrutura de Tabelas

<caption> - Legenda

Fornece um título ou descrição para a tabela.

<caption>Lista de Funcionários por Departamento</caption>

Deve ser o primeiro elemento dentro de <table> para melhor acessibilidade.

<thead>, <tbody> e <tfoot>

Agrupam linhas em cabeçalho, corpo e rodapé da tabela.

<table>
    <caption>Vendas Mensais</caption>
    <thead>
        <tr><th>Mês</th><th>Vendas</th></tr>
    </thead>
    <tbody>
        <tr><td>Janeiro</td><td>R$ 1000</td></tr>
    </tbody>
    <tfoot>
        <tr><td>Total</td><td>R$ 1000</td></tr>
    </tfoot>
</table>

Esses elementos melhoram a acessibilidade e permitem estilização diferenciada.

<colgroup> e <col>

Permitem aplicar estilos a colunas inteiras sem repetir em cada célula.

<colgroup>
    <col span="2" style="background-color: #f5f5f5;">
    <col style="background-color: #e9e9e9;">
</colgroup>

Tabelas Acessíveis

Para criar tabelas acessíveis, siga estas práticas recomendadas:

Exemplo de Tabela Acessível

<table>
    <caption>Horário de Funcionamento</caption>
    <thead>
        <tr>
            <th scope="col">Dia</th>
            <th scope="col">Abertura</th>
            <th scope="col">Fechamento</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <th scope="row">Segunda a Sexta</th>
            <td>09:00</td>
            <td>18:00</td>
        </tr>
        <tr>
            <th scope="row">Sábado</th>
            <td>10:00</td>
            <td>14:00</td>
        </tr>
    </tbody>
</table>

Use <th> e scope

Identifique claramente cabeçalhos e seu escopo (coluna ou linha).

Adicione <caption>

Forneça um título descritivo para a tabela.

Estruture com <thead>, <tbody>, <tfoot>

Agrupe linhas semanticamente para melhor organização.

Evite células mescladas complexas

colspan e rowspan podem dificultar a acessibilidade.

Tabelas Responsivas

Técnicas para fazer tabelas funcionarem em dispositivos móveis:

Método 1: Scroll Horizontal

.table-container {
    overflow-x: auto;
    max-width: 100%;
}

Método 2: Reformatação

@media (max-width: 600px) {
    table, thead, tbody, th, td, tr {
        display: block;
    }
    
    thead tr {
        position: absolute;
        top: -9999px;
        left: -9999px;
    }
    
    td {
        position: relative;
        padding-left: 50%;
    }
    
    td:before {
        position: absolute;
        left: 6px;
        content: attr(data-label);
        font-weight: bold;
    }
}

Combine com data-label nas células <td>:

<td data-label="Nome">João Silva</td>
