Estrutura Básica do HTML

Entenda os elementos fundamentais que compõem todo documento HTML5.

40 min de leitura Atualizado em: 15/08/2025

Introdução

Todo documento HTML5 possui uma estrutura básica composta por elementos essenciais que servem como base para a construção de páginas web. Esta estrutura é reconhecida pelos navegadores e é fundamental para a renderização correta do conteúdo.

O HTML5 trouxe simplificações na declaração de documentos em comparação com versões anteriores do HTML e XHTML.

Elementos Fundamentais

- Declaração do Tipo de Documento

A declaração deve ser a primeira linha em qualquer documento HTML. Ela informa ao navegador qual versão do HTML está sendo utilizada.

<!DOCTYPE html>

Características:

  • Não é uma tag HTML, é uma instrução para o navegador
  • No HTML5, a declaração é simplificada
  • Deve ser escrita em letras maiúsculas para compatibilidade

- Elemento Raiz

O elemento <html> é o container que engloba todo o conteúdo da página HTML, exceto a declaração .

<html lang="pt-BR">
    <!-- Todo o conteúdo da página -->
</html>

Atributos Importantes:

  • lang - Define o idioma principal do documento (importante para acessibilidade e SEO)

- Cabeçalho do Documento

O elemento <head> contém metadados sobre o documento, como título, links para folhas de estilo, scripts e outras informações que não são exibidas diretamente na página.

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>
    <link rel="stylesheet" href="estilos.css">
</head>

Elementos Comuns Dentro de <head>:

  • <title> - Define o título da página (exibido na aba do navegador)
  • <meta> - Fornece metadados sobre o documento
  • <link> - Usado para vincular recursos externos como CSS
  • <style> - Para CSS interno
  • <script> - Para código JavaScript

- Corpo do Documento

O elemento <body> contém todo o conteúdo visível da página web, incluindo textos, imagens, links, tabelas, listas, etc.

<body>
    <h1>Meu Primeiro Cabeçalho</h1>
    <p>Meu primeiro parágrafo.</p>
</body>

Características:

  • Só pode haver um elemento <body> por documento
  • Contém todos os elementos visíveis da página
  • Pode conter atributos para manipulação via JavaScript

Exemplo Completo

Aqui está um exemplo completo de um documento HTML5 básico:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Minha Primeira Página</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            line-height: 1.6;
            margin: 0;
            padding: 20px;
        }
    </style>
</head>
<body>
    <h1>Bem-vindo ao Meu Site</h1>
    <p>Este é um parágrafo de exemplo em minha primeira página HTML.</p>
    <script>
        console.log('Página carregada!');
    </script>
</body>
</html>

Boas Práticas

Declaração DOCTYPE

Sempre inclua <!DOCTYPE html> como a primeira linha do seu documento para garantir que o navegador renderize no modo padrão.

Atributo lang

Defina sempre o atributo lang no elemento <html> para melhorar a acessibilidade e SEO.

Metadados Essenciais

Inclua sempre charset="UTF-8" e a meta tag viewport para garantir renderização adequada em dispositivos móveis.

Estrutura Limpa

Mantenha uma estrutura bem organizada com indentação consistente para melhor legibilidade e manutenção.
