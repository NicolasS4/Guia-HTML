Elementos Interativos

HTML5 introduziu novos elementos para criar conteúdo interativo sem depender de JavaScript.

Elementos de Interatividade

<details> e <summary>

Criam um widget de revelação que o usuário pode expandir/recolher.

<details>
    <summary>Mais Informações</summary>
    <p>Conteúdo detalhado que será mostrado quando expandido.</p>
</details>

Atributos:

  • open - Mostra o conteúdo inicialmente expandido

<dialog> - Caixa de Diálogo

Representa uma caixa de diálogo ou subjanela interativa.

<dialog open>
    <p>Esta é uma caixa de diálogo aberta!</p>
    <button onclick="this.parentNode.close()">Fechar</button>
</dialog>

<button onclick="document.getElementById('dialog').showModal()">
    Abrir Diálogo
</button>

<dialog id="dialog">
    <p>Conteúdo do diálogo modal.</p>
    <button onclick="this.parentNode.close()">Fechar</button>
</dialog>

Métodos:

  • show() - Mostra diálogo não-modal
  • showModal() - Mostra como modal (bloqueia outros conteúdos)
  • close() - Fecha o diálogo

Exemplos Práticos

FAQ com <details>

O que é HTML5?

HTML5 é a quinta versão principal da linguagem HTML, trazendo novos elementos, APIs e melhorias para criação de aplicações web modernas.

Quais navegadores suportam esses elementos?

Todos os navegadores modernos suportam <details> e <dialog>. Para navegadores mais antigos, pode ser necessário um polyfill.

Elementos Obsoletos

Alguns elementos interativos antigos foram descontinuados no HTML5:

<menu> (antigo)

Originalmente destinado a menus de comandos, foi removido e depois reintroduzido com semântica diferente.

Evite usar o elemento <menu> antigo. Para menus de navegação, prefira listas (<ul>) estilizadas.

Melhores Práticas

Use JavaScript para interações complexas

Elementos nativos são limitados. Para experiências ricas, combine com JavaScript.

Forneça alternativas acessíveis

Garanta que conteúdo interativo seja acessível via teclado e leitores de tela.

Teste em vários navegadores

Alguns elementos podem ter comportamentos ou estilos diferentes entre navegadores.
